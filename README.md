# WordMaster

WordMaster is a free, mobile-first collection of five browser word games:

- WordChain
- Anagram Vault
- CrossClue
- WordShrink
- LetterLock

Progress and local leaderboard data are stored per player in `localStorage`. The app has no login, payment flow, backend, or runtime API dependency.

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```

Built with React, Vite, plain JavaScript, and handwritten CSS.
