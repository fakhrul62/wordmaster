# Retention Systems

WordMaster now keeps long-term engagement state at the player profile level.

- Every mini-game owns an independent level track and high score.
- Daily login streaks award coins, XP, and points.
- Daily challenges rotate from the game catalog and reward coins, XP, and points.
- Special events rotate twice per month with limited missions and event currency.
- Achievements unlock profile titles for dashboard and leaderboard display.
- The dashboard shows current progress, rewards, milestones, events, achievements, and leaderboard position.

The current implementation stores progress locally and preserves the same profile fields through optional account sync.
